FONTLOG for Barrio font

This file provides detailed information on the Barrio Font Software.
This information should be distributed along with the Barrio Font and any derivative works.

Basic Font Information

Barrio is a typeface family that supports Unicode language range: Latin-1 Supplement, Latin Extended-A

To contribute to the project contact Omnibus-Type at omnibus.type@gmail.com
